Etiquetas HTML

Este ejemplo muestra como combinar algunas de las etiquetas básicas de HTML.
Recuerda que es importante entender las diferencias entre ellas.

Etiqueta ul+li

Si listamos personas nominadas a los Oscars, dado que el orden no altera el significado.
debemos usar ul.

  • David Verdaguer
  • Jesús Castro
  • Israel Elejalde
  • Dani Rovira

Etiqueta ol+li

En el caso de que estemos listando elementos donde el orden es importante, como por
ejemplo la clasificación de un mundial de fútbol, debemos usar ol.

  1. España
  2. Paises Bajos
  3. Alemania
  4. Uruguay
